# aws-infra

Учебная Terraform-конфигурация для курса Git Bootcamp (день 6): VPC, EC2 для WordPress, security groups.

> **Дисклеймер.** В реальном проекте конфигурацию разбили бы на модули (`modules/network`, `modules/compute` и т.д.).
> Здесь всё в корне репозитория — чтобы история коммитов оставалась наглядной.
>
> `terraform apply` **запускать не нужно** — задание про Git, не про AWS.
